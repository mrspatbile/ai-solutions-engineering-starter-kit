# AI Design Review Checklist

- [ ] Why is AI needed?
- [ ] Could deterministic code solve the task?
- [ ] What data is sent to the model?
- [ ] Where does context come from?
- [ ] How is retrieval evaluated?
- [ ] What happens when retrieval fails?
- [ ] Can the model state that evidence is missing?
- [ ] How is output validated?
- [ ] How are numerical calculations handled?
- [ ] How is prompt injection addressed?
- [ ] How is cost controlled?
- [ ] How would production monitoring work?
- [ ] Where is human review required?
